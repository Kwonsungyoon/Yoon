HyperText Mark-up Language

"Hyper Text Mark-up Language" 의 약자. 웹 페이지의 모습을 기술하기 위한 규약. 프로그래밍 언어가 아니라 마크업 언어다. 헷갈리지 않도록 하자.웹사이트에서 흔히 볼 수 있는 htm이나 html 확장자가 바로 이 언어로 작성된 문서다.
HTML은 문법 오류에 관대한 편이다. 그래서 닫는 태그를 누락한다든가 태그에 오타가 난다든가 하는 오류가 발생해도 어느 정도는 씹어먹고 작동할 수 있다. 물론 '어느 정도'까지만.

CSS

CSS는 HTML 본연의 기능을 극대화하고 CSS를 통해서 HTML의 가독성과 효율을 높이기 위해서 등장했다.
CSS를 적용시키는 방법 중 하나는 style태그이고 다른 한가지는 속성을 이용하는 것이다. 태그를 이용하는 것은 우리가 평소에 하던 것과 같지만 속성의 예시는 이것이다.
